# LeetCode Java

leetcode，剑指offer题解以及思路，同时也存放一些算法的理论知识。

**目录内容介绍**

```
algorithms/
├── LICENSE
└── theory              算法理论部分
├── practice            算法刷题部分
│   ├── for-the-offer       剑指offer
│   ├── leetcode            leetcode刷题
```
